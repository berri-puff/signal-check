***README*** 

## Getting Started

### Dependencies

- Node.js and NPM

### Installing

1. Clone the repository:
   ```
   git clone https://github.com/berri-puff/signal-check.git
   ```
2. Install dependencies:
   ```
   npm install
   ```

### Running the App

- To preview on a local port:
  ```
  npm run dev
  ```

### Unfinished enhancements / Trade-offs
Due to time pressure, there are a few unfinished features on this app. 
Such as:
  - Lack of mobile styling/responsiveness. Currently, this app is optimised to run on a web browser 
  - One of core UI was to `indicate when it belongs to a currently highlighted valid chain`. However, I have ultimiately made the decision to only display the valid chains if and when clicking on a 0. 
  - Lack of testing for the functionalities to ensure the correct results are returned, whether as a unit testing or as integration testing.
